## Milestone Summary

### Key Decisions

- **Wrapper-first strategy** (Phase 10) — V8 preserved through all phases; default flipped to v2 only in Phase 13
- **Lazy V8 imports** in v2_topology/_node-base.js — works around pre-existing V8 hmac_node.js CommonJS/ESM bug without touching V8 baseline
- **All 16 nodes native by Phase 12** — accelerated from original 4-phase plan; no Phase 11-12 parallelization needed since each phase was small enough
- **Canonical cleanliness enforced via lint** (Phase 13) — word-boundary regex prevents "audio" → "udio" false positives; model names isolated to dated annex
- **LLM-creative patterns combined** (Phase 14) — Patterns 1+3+5 in creative_source; Patterns 2+3+4+5+6 in screenplay per §3.3

### Issues Resolved

- V8 → v2.0 topology migration without breaking webhook + Telegram notifications (via wrapper + KAI_PIPELINE_MODE=v8 fallback)
- Capability-spec drift prevention (lint check + dated annex)
- Cross-cutting invariant propagation (InvariantBus replaces V8 JSON asset bus)
- Theory_critic consultative edge implemented as non-blocking (META-06 creator-pulled via V2Pipeline.invokeTheoryCritic)
- Pre-existing V8 hmac_node.js bug worked around without V8 baseline changes (lazy imports)

### Issues Deferred (Future Milestones)

- Live statistical GO/NO-GO (FUTURE-08) — requires hermes-agent v3.0 + actual budget
- V8 baseline complete removal (FUTURE-K2) — after live run validation + 6+ month deprecation period
- Cross-repo ADR tooling (FUTURE-K3) — v2.0 uses manual ADR
- Multi-form parameterization production-grade (FUTURE-K4) — v2.0 implements single-form
- GPU Runtime Manager V6 (FUTURE-K5) — v2.0 keeps V5.1

### Technical Debt Incurred (Non-blocking)

- ConsistencyContext.validate() implements 1 of 5 logic-critic checks (4 stubs)
- creative_source._scoreNovelty() uses heuristic; trope-catalog embedding database does not exist
- audio_pipeline composer sub-step falls through to stub if V8 BGM strategy API mismatches
- Pre-existing V8 bug (lib/gold-team-client.js → shared/hmac_node.js) documented but not fixed
