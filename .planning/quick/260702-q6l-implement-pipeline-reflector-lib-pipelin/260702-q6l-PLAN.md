| LLM (callLLM) → reflect() | Untrusted text output that must be JSON-parsed |
| .pipeline-assets files → reflector | Local files written by prior pipeline runs |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-q6l-01 | Tampering | workdir path in /run, /approve, /reject | mitigate | zod-validate as non-empty string; reject paths containing `..` or starting with `/etc`/`/usr`; resolve against a configured allow-root |
| T-q6l-02 | Injection | node `-e` subprocess script string | mitigate | build script from constants + JSON.stringify(args) only; NEVER string-interpolate raw user input; pass user values via `process.argv` or `process.env` and read inside the subprocess |
| T-q6l-03 | Info Disclosure | LLM response echo'd to API caller | accept | operator-only API; reflection content is intentionally surfaced |
| T-q6l-04 | DoS | Large reflection-suggestions.jsonl | mitigate | readPendingSuggestions caps at 1000 rows; paginate if exceeded |
