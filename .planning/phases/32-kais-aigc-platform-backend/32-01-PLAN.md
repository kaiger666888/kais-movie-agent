3. **`GoldTeamClient` class** with:
   - Class constants: `DEFAULT_BASE_URL = "http://192.168.71.140:8002"`, `DEFAULT_TIMEOUT = 60.0` (GPU tasks are long), `DEFAULT_POLL_INTERVAL = 5.0`, `DEFAULT_POLL_TIMEOUT = 600.0` (10min, matching Node.js).
   - `__init__(self, *, base_url=None, api_key=None, timeout=None, transport=None)`:
     - `self._base_url = (base_url or os.environ.get("KAIS_GOLD_TEAM_URL") or self.DEFAULT_BASE_URL).rstrip("/")`
     - `self._api_key = api_key if api_key is not None else os.environ.get("KAIS_GOLD_TEAM_API_KEY")`
     - `self._timeout = timeout if timeout is not None else self.DEFAULT_TIMEOUT`
     - `self._client = httpx.Client(timeout=self._timeout, transport=transport)` — transport=None is real network; tests inject MockTransport.
     - `self._callback_base_url = os.environ.get("KAIS_CALLBACK_BASE_URL", "http://192.168.71.140:3000")`
     - `self._callback_secret = os.environ.get("KAIS_HMAC_SECRET_MA_GT", "")` — used only for `verify_callback()` (inbound GPU callbacks).
   - `_headers(self)` returns `{"Content-Type": "application/json"}` plus `{"X-API-Key": self._api_key}` if key set. (CRITICAL-FINDING-02.)
   - `_degrade(self, operation, reason)` returns `{"degraded": True, "client": "gold_team", "operation": operation, "reason": reason}` and logs at WARNING.
   - `_unwrap(self, response_json)` — Node.js ref returns `result.data`; if the JSON has a `data` key, return `data`; otherwise return the JSON as-is (defensive — service may evolve).
   - `_request(self, method, path, body=None)` — central HTTP wrapper:
     - Build URL = `f"{self._base_url}{path}"`.
     - Call `self._client.request(method, url, headers=self._headers(), json=body if body is not None else None)`.
     - On `httpx.ConnectError` / `httpx.TimeoutException`: return `self._degrade("request", str(exc))` (do NOT raise — degrade contract).
     - On HTTP >= 500: return `self._degrade("request", f"HTTP {response.status_code}")`.
     - On HTTP >= 400 and < 500: raise `GoldTeamError(f"HTTP {response.status_code}: {text[:200]}", status=response.status_code, url=url)`.
     - On 2xx: parse JSON; if parsing fails raise `GoldTeamError(f"Invalid JSON response: {text[:200]}", status=response.status_code)`. Return parsed JSON.
   - **Public methods (mirror Node.js ref):**
     - `submit_task(self, *, task_type, params, assets=None, priority=5, description=None, callback_path=None)`:
       - Build callback_url: `f"{self._callback_base_url}{callback_path or '/callback/gpu-task'}"`.
       - Body: `{"task_type": task_type, "params": params, "assets": assets or [], "priority": priority, "description": description, "callback_url": callback_url, "callback_secret": self._callback_secret}`.
       - `result = self._request("POST", "/api/v1/tasks", body)`. If `result.get("degraded")`: return result. Otherwise return `self._unwrap(result)` (extract `task_id`, `state`, `created_at`).
     - `get_task(self, task_id)` → `self._unwrap(self._request("GET", f"/api/v1/tasks/{task_id}"))`.
     - `list_tasks(self, *, state=None, task_type=None, limit=20, offset=0)`:
       - Build query string from non-None filters.
       - `self._unwrap(self._request("GET", f"/api/v1/tasks?{query}"))`.
     - `wait_for_task(self, task_id, *, poll_interval=None, timeout=None)`:
       - Poll loop: `while time.monotonic() - start < timeout: task = self.get_task(task_id); if task.get("state") == "done": return task; if task.get("state") == "failed": raise GoldTeamError(f"GPU task failed: {task.get('error', '')}", status=None); time.sleep(poll_interval)`. After timeout: raise `GoldTeamError(f"GPU task timeout: {task_id}")`.
       - Defaults: `poll_interval or self.DEFAULT_POLL_INTERVAL`, `timeout or self.DEFAULT_POLL_TIMEOUT`.
       - NOTE: this method MAY raise — degrade is per-call (submit_task); if you've already got a task_id, a poll failure is a real failure, not a degrade.
     - `submit_task_degraded(self, **kwargs)` — convenience: wraps `submit_task` in try/except, returns `self._degrade("submit", str(exc))` on `GoldTeamError` or any other Exception. This matches Node.js `submitTaskDegraded`.
     - `verify_callback(self, body, signature)` — inbound HMAC verification for GPU task callbacks. Use `hmac.new(self._callback_secret.encode(), body.encode() if isinstance(body, str) else body, hashlib.sha256).hexdigest()` then `hmac.compare_digest(f"sha256={expected}", signature)`. Return bool. (Mirrors Node.js `verifyCallback`.)
     - `ping(self, timeout_ms=5000)` — health check: try `httpx.Client(timeout=timeout_ms/1000).get(f"{self._base_url}/health")`; return `response.is_success`. On any exception return False.
   - `close(self)` → `self._client.close()`. `__enter__`/`__exit__` for context manager.
