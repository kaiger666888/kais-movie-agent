**Key v5.0 hardenings vs Node.js ref:**
- **JWT bearer auth** (CRITICAL-FINDING-03): Node.js `submitReview` and `queryReviewStatus` do NOT attach any Authorization header. v5.0 REQUIREMENTS GPU-DIRECT-02 requires JWT bearer. Python port generates an HS256 JWT (using `PyJWT[crypto]==2.13.0` already in pyproject.toml) signed with `KAIS_REVIEW_JWT_SECRET`, valid 5 minutes, attached as `Authorization: Bearer <jwt>` to every outbound request when secret is set.
- **HMAC 5-minute timestamp window** (CRITICAL-FINDING-04 + D-11): Node.js `bin/callback-server.js` `verifyHmac()` is plain body HMAC match. v5.0 REQUIREMENTS GPU-DIRECT-02 requires 5min timestamp window. Python port adds `verify_callback(body, signature, timestamp)` that checks `abs(now - timestamp) > 300` BEFORE computing HMAC.

Output: 2 files (client module + test file). ~150-300 LOC client + 8-10 tests.
