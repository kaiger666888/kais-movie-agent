Cosine similarity formula (pure stdlib):
```python
def _cosine_similarity(vec_a: list[float], vec_b: list[float]) -> float:
    """Cosine similarity = dot(a,b) / (||a|| * ||b||). Returns 0.0 if either norm is 0."""
    dot = sum(a * b for a, b in zip(vec_a, vec_b))
    norm_a = math.sqrt(sum(a * a for a in vec_a))
    norm_b = math.sqrt(sum(b * b for b in vec_b))
    if norm_a == 0 or norm_b == 0:
        return 0.0
    return dot / (norm_a * norm_b)
```

Jaccard similarity formula (pure stdlib):
```python
def _jaccard_similarity(list_a: list[str], list_b: list[str]) -> float:
    """Jaccard = |A ∩ B| / |A ∪ B|. Returns 0.0 if both lists empty."""
    set_a = set(list_a)
    set_b = set(list_b)
    union = set_a | set_b
    if not union:
        return 0.0
    return len(set_a & set_b) / len(union)
```

Structure → numerical vector mapping (CONTEXT.md "Cosine similarity over numerical fields"):
```python
def _structure_to_numerical_vector(structure: dict) -> list[float]:
    """Extract the 3-dim numerical vector from a structure dict.

    Vector: [hook_position_sec, mean(turning_points_sec), emotion_drop_level]
    - hook_position_sec: int → float (direct)
    - mean(turning_points_sec): mean of the list (or 0.0 if empty)
    - emotion_drop_level: int → float (direct)
    """
    hook = float(structure.get("hook_position_sec", 0))
    tp_list = structure.get("turning_points_sec", [])
    tp_mean = float(mean(tp_list)) if tp_list else 0.0
    drop = float(structure.get("emotion_drop_level", 0))
    return [hook, tp_mean, drop]
```
