# 场景设计提示词

## 基础场景提示

```
设计一个电影场景的空间布局。

地点：{location}
时代：{era}
情感氛围：{mood}
艺术方向：{art_direction}

请提供：
1. 空间分区（3-6个功能区域，包含相对位置关系）
2. 每个区域的视觉特征和功能
3. 区域之间的动线关系
4. 关键道具和陈设
5. 空间隐喻（场景如何服务于叙事主题）
```

## 机位设计提示

```
为以下场景设计 3-5 个核心机位：

场景：{location}
风格：{art_direction}
氛围：{mood}

每个机位需包含：
- 机位名称（英文）
- 空间坐标 (x, y, z)
- 朝向目标
- 镜头焦段
- 拍摄意图（叙事功能）

要求：
- 至少1个全景建立镜头
- 至少1个中景叙事镜头
- 至少1个特写/情绪镜头
- 考虑角色在空间中的运动轨迹
```

## 氛围设计提示

```
为场景设计氛围层次：

地点：{location}
时代：{era}
情绪：{mood}
光线质感：{art_direction}

请描述：
1. 主光源（方向、色温、强度）
2. 辅助光源（填充、轮廓、环境光）
3. 色彩情绪（主色调 + 点缀色）
4. 材质与纹理（空间表面的视觉质感）
5. 空气感（烟雾、灰尘、雾气等大气效果）
6. 声场暗示（空间的声音特征）
```
